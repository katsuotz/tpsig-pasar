# TPSIG Pasar

Web Apps untuk melihat lokasi pasar melalui Maps. Dibuat untuk Tugas Akhir Mata Kuliah TPSIG.

## Requirements

- PHP >= 7

## Installation

- Import database dari tpsig_pasar.sql
- Atur konfigurasi database pada config.php

## Tech

- PHP
- Leaflet - Maps JavaScript Library
